## 배열
일정한 차례, 간격  
차례(순서)와 관련된 기본적 자료구조

- 메모리 공간(메인 메모리, DDR)의 물리적 위치를 '순서적'으로 결정
- 원소 값의 물리적 순서
- 같은 자료형, 같은 크기의 기억공간
- 인덱스로 직접 접근

A[ ]의 시작주소를 a라고 가정하면 A[i]의 주소는 [a+i*k]  

### 행우선
```
for (...i...)
    for (...j...)

/* 
    A[i,j]
    [0,0]
    [0,1]
    [1,0]
    [1,1]
메인메모리에 저장되는 순서 그대로
*/
```
열우선
```
for(...j...)
    for (...i...)

/*
    A[j, i]
    [0,0]
    [1,0]
    [0,1]
    [1,1]
메인메모리에서는 크게 문제가 되지 않지만 secondary 메모리
(하드디스크)일때는 track이나 실린더를 바꾸는 현상발생. 데이터의 I/O 시간 지연 발생
*/            
```
